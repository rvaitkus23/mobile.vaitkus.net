Used framworks and libraries
This sample application is build following best practices of hybrid apps development using Cordova (a.k.a. PhoneGap) tool:
  • AngularJS framework:
    • SPA - Single Page Application
    • Templates
    • Structured MV* architecture
    • Routing
  • Hidden HTMLish behavior
  • Architect for Performance:
    • Don't generate UI on the server
    • Don't wait for data to display the UI
    • Cache everything (data, selectors, precompiled templates, ...)
    • Use Hardware acceleration
    • Avoid click event's 300ms delay
    • Use CSS sprite sheets
    • Limit shadows and gradients
    • Avoid reflows
    • Test
Ionic framwork is used as as main framework and it implements AngularJS and all of other best practices in the list. Added few custom directives to implement some functionality and custom UI.
